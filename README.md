# 🌿 Environmental Science (EVS) Quiz App

![React](https://img.shields.io/badge/react-%2320232a.svg?style=for-the-badge&logo=react&logoColor=%2361DAFB)
![Vite](https://img.shields.io/badge/vite-%23646CFF.svg?style=for-the-badge&logo=vite&logoColor=white)
![Framer Motion](https://img.shields.io/badge/Framer-Black?style=for-the-badge&logo=framer&logoColor=blue)

A beautiful, fully responsive **Environmental Science MCQ Quiz Application** built using **React (Vite)**. It features a gorgeous **Soft Neumorphism UI Design**, complete with smooth page transitions, automatic dark mode support, and interactive animations. 

Test your knowledge across 200 hand-picked questions covering the core chapters of Environmental Science!

---

## ✨ Features

- **🎨 Stunning Neumorphic Design**: Custom CSS modules implementing soft, realistic inner and outer shadows that look incredible in both light and dark modes.
- **📚 200 Comprehensive Questions**: Contains 4 distinct chapters:
  - 🌍 Environmental Science Basic Concepts
  - 🦁 Ecosystems and Biodiversity
  - 💧 Natural Resources
  - 🏭 Environmental Pollution
- **🎯 Smart Chapter Selection**: Users can choose to be tested on specific chapters, or select "All" for a completely randomized quiz across all topics.
- **⏱️ Timed Questions**: Each question has a 30-second countdown timer. If the timer runs out, it automatically advances.
- **📊 Instant Feedback & Analytics**: See immediately if your answer was correct with green/red indicator animations. At the end, receive a detailed breakdown of your performance, letter grade, and a perfect-score confetti celebration!
- **📱 Fully Responsive**: Implemented mobile-first layouts so the app looks stunning on tiny phones, tablets, and massive desktop monitors.

## 🚀 Quick Start (Local Development)

To run this project locally, follow these simple steps:

### 1. Clone the repository
```bash
git clone https://github.com/priyabratasahoo780/EVS.git
cd EVS
```

### 2. Install dependencies
```bash
npm install
```

### 3. Start the development server
```bash
npm run dev
```

The app will now be running on `http://localhost:5173`.

## 📦 Deployment

This project uses Vite, making it incredibly simple to deploy to modern hosting platforms.

### Vercel (Recommended)
This repository includes a `vercel.json` file. Simply import the repository into your Vercel dashboard and hit deploy. It will automatically detect Vite and run `npm run build`.

### Netlify
Run `npm run build` locally, and drag the generated `dist` folder into Netlify Drop, or link your GitHub repository to Netlify for automatic deployments.
